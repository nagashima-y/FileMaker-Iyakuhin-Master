FileMaker-Iyakuhin-Master
=========================

[診療情報提供サービス](http://www.iryohoken.go.jp/shinryohoshu/)（医薬品マスター）、[Medis](http://www.medis.or.jp/)（医薬品HOTコードマスター）からダウンロード出来る一般公開情報を利用した医薬品マスター検索（FileMaker Pro 12以降）

・後発品
・医薬品名
・基本漢字名称
・点数
・剤形
・注射容量
・製造会社
・販売会社
・JANコード
